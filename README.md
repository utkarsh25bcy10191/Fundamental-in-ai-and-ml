PROJECT TITLE:
Covid Detection Expert System using PROLOG

PROJECT OVERVIEW:
This project is a simple expert system built using PROLOG, designed to help users identify the presence of COVID-related symptoms.The system interacts with the user, ask for their name, and then presents a list of symptoms. Based on the user's selected symptoms, it provides feedback and suggests whether they should seek medical attention.

Features:
1. User interactions:
   Ask's the name
   greet user and start evaluation automatically
2. Symptoms list:
   A list of 10 symptoms
3. Knowledge base:
   contain symptom: fever, headache etc
4. Error handling:
   Detect invalid choices
5. Begineer-friendly:
   Demonstrate AI concepts like rule-based inference

TECHNOLOGIES USED:
PROLOG(SWI-PROLOG)
Basic text editor
Command-line interface
